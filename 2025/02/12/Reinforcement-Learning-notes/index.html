增量式更新

马尔可夫奖励过程MRP

由四部分组成 < S, P, r, γ>

  • 状态集
  • 状态转移矩阵
  • r(s)转移到s状态可获奖励的期望
  • 衰减因子 惩罚未来获得的奖励

回报Gt = Rt + γRt + 1 + γ2Rt + 2 + ⋯ = Σk = 0∞γkRt + k

价值 价值函数V(s) = 𝔼(Gt|St = s)

展开上式得贝尔曼方程(Bellman equation) V(s) = r(s) + γΣs′ ∈ SP(s′|s)V(s′)

采用矩阵表示 所有状态价值V = [V(s1), V(s2), ⋯]T，奖励R = [r(s1), r(s2), ⋯]T，则有 V = R + γPV

价值有解析解V = (I − γP)−1R 该解析解在计算上有O(n3)复杂度

马尔可夫决策过程 MDP

 < S, A, P, r, γ>

A:动作的集合。P变成P(s′|s, a)同时r变成r(s, a)

引入策略π(s, a) = P(At = a|St = s)

最为重要的两个函数

  • 状态价值函数Vπ(s) = 𝔼[Gt|St = s]
  • 动作价值函数Qπ(s, a) = 𝔼[Gt|St = s, At = a]

两者有关系Vπ(s) = Σa ∈ Aπ(a|s)Qπ(s, a)，该式即指在状态s下采用所有行动的概率与该行动的价值函数相乘之和，也是期望的一种形式。反过来说，某动作也可采用前述贝尔曼方程展开得到采用V的表示，即Qπ(s, a) = r(s, a) + γΣa ∈ AP(s′|s, a)Vπ(s′)

贝尔曼期望方程 Bellman Expectation Equation

可以看出上式第二个式子即为上面两个关系合起来。

另外可以采用边缘化(marginalization)的方法将MDP转化为MRP（将一个状态和动作合在一起定义为一个新的状态从而取消掉动作）。

要解决的目标问题：计算Vπ(s) 可以使用蒙特卡洛算法：

  • 采样若干序列
  • 更新s的计数器(N)和总回报(M)
  • 每个状态进行估计V=M/N

上述过程借鉴多臂老虎机可以使用增量式更新

讨论策略：占用度量

不同的策略会使智能体访问到不同的概率分布 定义下面两个变量以讨论策略

  • 状态访问分布 state visitation distribution Vπ(s) = (1 − γ)Σt = 0∞γtPtπ(s)
  • 占用度量 occupancy meassure ρπ(s, a) = (1 − γ)Σt = 0∞γtPtπ(s)π(a|s)

一个策略的占用度量和策略一一对应。不同策略对于同一状态动作对占用度量不同

最优策略 RL的目标得到最多期望回报

定义策略之间的偏序关系 π > π′ ⇔ ∀s, Vπ(s) > Vπ′(s) 最优状态价值函数 最优动作价值函数 贝尔曼最优方程 Bellman Optimality Equation

此后采用了动态规划 差分时序等进行了求解
