基础算法

动态规划

基于动态规划的强化学习算法适用于白盒环境，事先知道P（状态转移矩阵）与r（reward），不需要进行交互。

动态规划分为两步 - 策略迭代 - 策略评估 V(s) ← r(s, π(s)) + γ∑s′P(s′|s, π(s))V(s′) - 策略提升 π(s) ← argmaxar(s, a) + γ∑s′P(s′|s, π(s))V(s′) - 价值迭代 V(s) ← maxar(s, a) + γ∑s′P(s′|s, π(s))V(s′) 不显式更新策略π

价值迭代采用了Bellman Optimality Equation

时序差分

TD: temporal difference基于时序差分的强化学习算法 并不事先知道P，需要和环境交互 - Sarsa 名字由来为用到了s, a, r, s, a′ - Q(s, a) ← Q(s, a) + α[r + γQ(s′, a′) − Q(s, a)] - 采用ϵ − greedy策略选择a及a’ 防止有些动作从未出现过 - 更为保守 （悬崖上倾向于离悬崖更远走） - Q-learning - Q(s, a) ← Q(s, a) + α[r + γmaxQ(s′, a′) − Q(s, a)] - 只采用ϵ − greedy策略选择a而不选择a’

在这一节定义了在线和离线策略学习 - 在线策略学习 on-policy 自来水 sarsa - 离线策略学习 off-policy 脸盆 具有更小的样本复杂度 Q-learning

采用数据的策略为行为策略(behavior policy) 更新的策略为目标策略(target policy)

当行为策略与目标策略一致时即为在线策略 不一致时即为离线策略

Dyna-Q

基于模型但是环境模型通过采样估计得到 由Q-learning+Q-planning得到 略

进阶算法

DQN 深度强化学习的开端

DQN: Deep Q Network 前述方法均采用表格的形式记录Q 一个s,a对对应一个价值Q 仅适用于离散有限情况 采用神经网络表示Q 无限s,a对 由更新Q变为训练Q Q-Learning: Q(s, a) ← Q(s, a) + α[r + γmaxa′ ∈ AQ(s′, a′) − Q(s, a)]

即定义Q网络损失函数

损失可以粗略记成 L(w) = mean((Q_w(s_i,a_i) - y_i)^2) / 2。

其中

yi = ri + γmaxa′Qw−(si′, a′)

DQN还有两个关键模块 - 经验回放——使样本满足独立假设 - 非独立同分布的数据对训练神经网络影响大 - MDP交互数据：该时刻的状态与上时刻有关 - 事实上这一手段并没有完全解决独立同分布的问题 只是缓解 - 目标网络 - 解决不稳定问题——目标包括输出 - 自举效应（Bootstrapping） - 采用两个网络分别计算Qw(si, ai)与ri + γmaxa′ ∈ AQ(si′, a′))——训练网络Qw与目标网络Qw−

DQN的改进： - Double DQN - DQN对Q有过高估计overestimaion - 使动作选取依靠训练网络 将maxa′ ∈ AQw−(si′, a′)替换为maxa′ ∈ AQw−(si′, arg maxa′Qw(s′, a′)) - Dueling DQN - 定义优势函数 - 将V与A分别建模 - ### 策略梯度 由基于价值到基于策略 策略学习目标函数 J(θ) = 𝔼(Vπθ(s0))

策略梯度定理 ∇J(θ) ∝ ∑s ∈ Svπθ(s)∑a ∈ AQπθ(s, a)∇θπθ(a|s)

将V与Q的关系代入上式上式 = 𝔼[Qπθ(s, a)∇θlog πθ(a|s)]

采样与更新一致 属于在线策略

关键是要对Qπθ(s, a)进行估计 用蒙特卡洛方法进行估计——REINFORCE
